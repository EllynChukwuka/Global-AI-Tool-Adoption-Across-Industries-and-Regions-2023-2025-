Technical Report
Global AI Tool Adoption Across Industries and Regions (2023–2025)
Platform: Power Bi(Dashboard & Spreadsheet Analysis)

 
1. Objective of the Project
The objective of this project is to analyze global trends in AI tool adoption across countries, industries, age groups, and AI platforms from 2023 to 2025. The analysis aims to identify usage patterns, adoption intensity, and key drivers of AI growth using Power BI for data analysis and dashboard development.

Problem Being Addressed
With rapid AI expansion, organizations and policymakers face challenges in understanding:
Which countries and industries are leading AI adoption
Which AI tools dominate daily usage
How demographics (age groups) influence adoption
Where strategic investments in AI infrastructure and training should be focused
This analysis addresses these challenges by transforming raw adoption data into actionable insights.

2. Key Datasets and Methodologies
Datasets Used:
AI tool usage data (ChatGPT, Midjourney, Stable Diffusion, Bard, Claude)
Adoption rates by country and industry
Daily active users (DAU)
Age-group-based adoption metrics
Methodologies Used in Excel:
Pivot Tables & Pivot Charts
Power BI Dashboards with KPI cards
Aggregation using SUM, COUNT, and AVERAGE
Sorting, filtering, and slicers
Data visualization (bar charts, line charts, treemaps)
 
3. Story of Data
Data Source
The dataset is derived from a simulated global AI adoption dataset, designed to represent real-world AI usage trends across multiple regions and industries between 2023 and 2025.

Data Collection Process
Data was collected through:

Aggregated platform-level usage metrics
Regional and industry adoption surveys
Modeled daily active user statistics
The dataset was consolidated into a single Power Bi for analysis.

Data Limitations or Biases

7. Post-Analysis and Insights
Key Findings
Total Daily Active Users: 731 million
Top AI Tool by DAU: ChatGPT (292M users)

Leading Age Group: 18–24
Top Industries: Technology, Education, Manufacturing

High-Adoption Countries: Australia, France, India, South Korea

Comparison with Initial Findings
Initial assumptions about tech-sector dominance were confirmed.
However, adoption in healthcare and agriculture was higher than expected, indicating emerging opportunities.
